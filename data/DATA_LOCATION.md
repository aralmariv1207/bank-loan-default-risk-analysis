# Data Location

Due to the significant file size (3M+ records), the primary dataset for this project is hosted on Google Drive.

**Dataset:** `order_products.csv`  
**Access Link:** [Download Dataset Here](https://drive.google.com/file/d/1Xl0OpxHlM-hD12Z3v-7uUeLcl0p-D7o8/view?usp=sharing)

### **Setup Instructions**
1. Click the link above to access the file on Google Drive.
2. Select the **Download** icon (top right). *Note: Google Drive cannot preview this file because it exceeds the size limit.*
3. Place the downloaded `.csv` file into the `/data/` directory of your local clone of this repository.
4. Ensure the file is named `order_products.csv` to maintain compatibility with the notebook paths.
